# Локальный музыкальный плеер

Этот проект создан в качестве альтернативы имеющимся плеерам которые не предоставляют необходимой возможности импорта музыки и минималистичночти интерфейса 

## Особенности проекта

- **Минималистичный дизайн**: Стильный, чистый интерфейс без лишних деталей, плавная плавающая навигация и поддержка тёмной/светлой тем.
- **Удобный импорт музыки**: Сканирование локальных каталогов через Android Storage Access Framework (SAF) с рекурсивным поиском файлов и быстрым кешированием метаданных в Room SQLite.
- **Интеллектуальное воспроизведение**:
  - Автоматическая пауза при входящих/исходящих звонках и возобновление проигрывания сразу после завершения вызова (Audio Focus).
  - Плеер с интерактивной вращающейся виниловой пластинкой, автоматически извлекающий обложки треков.
  - Интеграция с системной медиа-сессией (Media Session) для управления с экрана блокировки и шторки уведомлений.
- **Современный стек технологий**: Kotlin, Jetpack Compose, Room Database, Coroutines & Flow, MVVM-архитектура.

## Требования для сборки

- **Java Development Kit (JDK)**: Версия 17
- **Android SDK**: API level 24 (minSdk) / API level 36 (targetSdk)
- **Среда разработки**: Android Studio Koala / Ladybug или новее
- **Система сборки**: Gradle 8.x+ (используется Gradle Kotlin DSL `.gradle.kts`)

## Инструкция по сборке

Проект полностью готов к сборке и включает встроенный механизм автоматического восстановления ключа подписи из Base64-копии для обеспечения непрерывности CI/CD.

### Шаг 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd <папка-репозитория>
```

### Шаг 2. Сборка проекта

Вы можете выполнить сборку приложения через Android Studio или напрямую через командную строку.

#### Использование командной строки

Для сборки отладочной (Debug) версии APK выполните:

```bash
gradle assembleDebug
```

После завершения процесса скомпилированный APK-файл будет находиться по пути:
`app/build/outputs/apk/debug/app-debug.apk`

Для сборки релизной (Release) версии APK (подписанной ключом `my-upload-key.jks`):

```bash
gradle assembleRelease
```
